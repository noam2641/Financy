## FINDINGS
- **F-A2-01** yfinance default `auto_adjust=True` bakes future corporate actions into historical OHLC (target-level look-ahead). CRITICAL · REQ-PRICE-001/002, REQ-LAB-001, REQ-CONV-DATA-008 · blocks release_0_acceptance, release_1A, shadow, paper, live. External [B]: yfinance ≥0.2.28 defaults `auto_adjust=True`; back-adjustment retro-applies future splits/dividends. Correction: ingest `auto_adjust=False, back_adjust=False`; persist raw OHLC + separate versioned corporate-action table; derive adjusted on demand with `corporate_action_version`/`split_adjustment_factor`. Labels MUST declare `price_basis`.
- **F-A2-02** yfinance exposes no vintages/ingestion timestamps → cannot populate bitemporal fields or drive SYSTEM_REALISTIC replay. CRITICAL · REQ-DATA-001, REQ-TIME-003/006/007 · blocks architecture_freeze, shadow, paper, live. `system_available_at` is a property of the system's ingestion event, nonexistent for any pre-system date. Correction: pre-go-live backtests INFORMATION_THEORETIC only (REQ-TIME-009 labeled); SYSTEM_REALISTIC available only forward; `is_estimated_timestamp=true` for any estimate.
- **F-A2-03** Yahoo corporate-action timing/repair unreliable; no `corporate_action_version`. HIGH · REQ-PRICE-004, REQ-CONV-DATA-003/008. External [B]: documented yfinance split-not-applied and wrong weekly/monthly dividend-adjustment issues; non-deterministic across pulls. Correction: version/freeze corporate-action set; reconcile against independent source.
- **F-A2-04** Delisted tickers vanish from Yahoo/yfinance → survivorship bias; no terminal-event fields. CRITICAL · REQ-LAB-013, REQ-DOM-016/017 · blocks release_0_acceptance, release_1A, final_holdout. External [C]: free single-ticker feeds don't serve full delisted history. Correction: `instrument_master` with `delisting_date/reason/successor_instrument_id` + terminal-event returns; source delistings independently; forbid universe from live-ticker resolution alone.
- **F-A2-05** PIT GICS sector membership is proprietary and NOT free → the DEFAULT primary target `sector_excess_return_h` is un-sourceable PIT for free, and contradicts R0/R1A sector exclusion. CRITICAL · REQ-LAB-006/009/010, REQ-CONV-GRAPH-003, REQ-REL0-003, REQ-REL1A-002 · blocks architecture_freeze, release_0_definition, release_1A. External [A]: GICS is exclusive property of MSCI/S&P; PIT "GICS History" is a commercial product; no free official PIT GICS. External [C]: free membership proxies (EODHD constituents from 2000; Wikipedia; ETF holdings) are index membership not GICS sector, not an official PIT taxonomy. Correction options (freeze one): (1) re-designate R0/R1A primary to market-excess/absolute, mark sector-excess UNAVAILABLE (REQ-LAB-010 permits); (2) substitute free PIT **SIC** from SEC filings (`sector_taxonomy=SIC`, coarser but genuinely PIT); (3) license GICS History.
- **F-A2-06** SEC EDGAR CAN deliver first-print vs restated PIT (free, official) — but the free `companyfacts` API returns latest-restated by default. MEDIUM (R2+) · REQ-DATA-005/006/007 · blocks release_2A. External [A]: EDGAR submissions carry `ACCEPTANCE-DATETIME`/`filedAt`; amendments (10-K/A, 10-Q/A) distinct filings. External [B]: `companyfacts` returns all reported values incl. restated; naive dedup keeps latest-filed. Correction: build bitemporal fundamentals store from as-filed submissions keyed on `accession_number` + `filing_accepted_at`; never overwrite.
- **F-A2-07** FRED default returns latest-revised (look-ahead); ALFRED provides free first-release vintages. HIGH · REQ-DATA-001, REQ-TIME-008, REQ-CONV-GRAPH-005 · blocks release_2A, architecture_freeze (policy). External [A/B]: ALFRED archives real-time periods & vintages; omitting `realtime_*` defaults to today → fully-revised. Correction: mandate ALFRED with `realtime_start`/`vintagedates` for historical macro; store vintage as `provider_published_at`/`valid_from`.
- **F-A2-08** Free news gives publication time but not system-availability time; licensing restricts persistence/redistribution. HIGH (R2A; DEFERRED R0/R1A) · REQ-DATA-004, REQ-TIME-003, REQ-SCH-035 · blocks release_2A, shadow/live (licensing). External [C]: GDELT redistribution scrutiny; NewsAPI-class free tiers forbid commercial/production; none supply system-availability time. Correction: store publication time as `publicly_available_at`; leave `system_available_at` null/estimated; complete licensing review before persisting bodies/features.
- **F-A2-09** Ticker is not a key; yfinance keys on current ticker only → entity-resolution corruption. HIGH · REQ-DOM-016, REQ-CONV-LINK-007 · blocks architecture_freeze, release_0_acceptance. - **F-A2-10** REQ-LAB-001 declares no `price_basis` → labels silently inherit yfinance back-adjusted basis (compounds F-A2-01). HIGH · REQ-LAB-001/007, REQ-PRICE-001/004 · blocks release_0_definition, release_0_acceptance.
